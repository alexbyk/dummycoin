DummyCoin infomration

Information

DummyCoin is a simple blockchain client-server implementation. I made it as a test task for "seaching for a new jobs" purposes

You can send coins from anyone to anyone. You can event send coins you don’t own (and got a minus balance) – that’s for simplification by design.

Dummy coin has a constant proof-of-the-work complexity, simplified coin tracks without encryption and simplified consensus algorithm.

While the algorithm is simple, the project itself is a good example of well designed, simple and testable code
